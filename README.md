# SoTA Finance Dashboard

Simulador interactivo en una sola página para proyectar crecimiento de catálogo (RL), clientes, ingresos, costos operativos y margen usando Chart.js.

## Requisitos
- Navegador moderno con acceso a internet (Chart.js se carga por CDN).

## Cómo ejecutarlo
1) Clona o descarga el repo.  
2) Abre `index.html` en el navegador.

## Controles principales
- Nuevas RL/mes, crecimiento y churn de clientes, precio por RL, densidad de compra y costo de mantenimiento por RL.

## Estructura
- `index.html`: interfaz, estilos y lógica del simulador.
- `media/`: íconos y logo usados en la UI.
